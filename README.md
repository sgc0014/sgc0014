## 💼 What I've Built

### 🤖 Ideabox AI — AI Workflow Builder
Configuration-driven UI platform with a **visual drag-and-drop agent flow builder** (think n8n). Built AI chatbot orchestration interfaces, multi-agent coordination flows, and maintained a monorepo frontend ecosystem used across multiple apps.

### 🛥️ Yacht Cloud — Multi-Tenant Reservation System
Full-featured yacht booking platform with multi-role access, SSR/ISR optimization, advanced filtering, and a polished landing page that converts.

### 🗑️ Waste Management System — Tel Aviv Municipality
RBAC web app providing real-time waste collection insights via interactive maps. Integrated Google Maps APIs for driver task assignment and built workplan flows for operational teams.

### ⚖️ Office Action Tool — Legal AI Assistant
Helped lawyers generate patent applications and office action responses with real-time AI streaming. Worked with prompt engineers to ensure smooth document analysis workflows.

### 🐦 Tweetsky — Twitter Growth Tool
Comprehensive Twitter thread manager with RSS-to-tweet, engagement analytics, hashtag generation, and scheduling — powered by Twitter APIs and AWS Lambda.

### 🎓 Glider — EdTech Platform
Consultancy-focused course builder enabling educators to create and deliver content in multiple formats (PDFs, videos, quizzes), built with Next.js.

### 🏆 GG Maidan — Esports Tournament Platform
Led architecture and development of a gaming platform with tournament creation, team management, secure payments, and live leaderboards — built with React and Firebase.

### 📸 Pixto — Photographer Workspace
Intuitive digital workspace for photographers with video uploading and client proofing UI.
